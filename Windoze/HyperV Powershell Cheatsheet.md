# Hyper V Powershell Cheatsheet

* List all VMs
	* `get-vm`

* Start a VM
	* `start-vm -name "name of vm"`

* Stop vm
	* `stop-vm -name "name of vm"`

* Compact VHD
	* `optimize-vhd -path "path to vhd file"`
