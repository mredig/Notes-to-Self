# Raspbian overscan settings

1. Edit /boot/config.txt
2. adjust values for overscan top, left, right, and bottom
3. reboot each time to see changes

Settings for RPi3 with Panasonic 720p display:

`overscan_left=-14`
`overscan_right=-14`
`overscan_top=-24`
`overscan_bottom=-24`
